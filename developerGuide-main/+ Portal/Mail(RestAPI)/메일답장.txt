[창업_차세대] 업무포털 <-> 메신저, 메일 시스템 연동 관련(2)

안녕하세요 아이티메이트 정성희 입니다. 

보내주신 내용 잘 받았습니다. 추가 문의 사항으로 연락 드립니다.

1. 메신저 안 읽은 쪽지 관련
 DB 쿼리 보내주셨는데, 접속 관련 한 정보가 필요 합니다.
 1) DBMS
 2) DBMS 버전
 3) SID or Service Name or 이중화 연결문자열 
 4) DB 접속 계정 정보 
 위 항목에 대한 개발기/운영기 에 각각의 정보.
 
보내주시면 연결 방안 검토 하겠습니다.


2. 내/외부 메일 REST API 관련
 참고 소스를 보내주셨는데, REST(URL api 로 표현하신) 의 경우 연동 문서가 필요 합니다.
  1) API URL
   - SSL 인경우 SSL 헤더 세팅 정보 
  2) HTTP 메소드 (GET/POST..)
  3) 파라미터 정보 (파라미터 이름, 값)
   - 보내주신 소스 참고 하겠지만 어떤 API 일때 어떤 값을 서버에서 받아야 하는지에 대해 프로토콜 에 따라 정의가 필요합니다.
  4) 결과값 정의 
   - 프로토콜에 따른 포지티브 상태시 출력 포맷 (예시가 있으면 좋고요)  
   - 공통 헤더 나 에러코드 정의 (포지티브 상태가 아닐때 예외처리를 위해)
